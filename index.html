HTML

  • Every HTML file must be declared on the first line.
  • The head element houses metatags, title, page title, and CSS links.
  • The body element contains the text and image content that appears on the page for users.

CSS

  • CSS files do not need to be declared on the first line like HTML files.
  • (*) is used to add properties to all items on a page.
  • Use the (.) operator before classes and (#) before ids.

Git

  • By using Git and Github, we can safely collaborate with peers on different parts of a project
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -a: adds all edited files to staging

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Loops allow us to complete repetitive tasks with a single code block.
